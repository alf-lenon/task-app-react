# React Task Manager

A simple task management application built with React.
This project is a recreation of my Vanilla JavaScript task app, rebuilt using React to practice component-based architecture and state management.

## Features

* Add new tasks
* Prevent empty task submission
* Delete tasks
* Dynamic task rendering using React state

## Technologies Used

* React
* JavaScript (ES6+)
* CSS
* Vite

## Future Improvements

* Mark tasks as completed
* Edit tasks
* Save tasks in localStorage
* Task filtering (All / Completed / Pending)

## Purpose

This project was created to practice React fundamentals such as:

* useState
* Component structure
* Props
* State updates
* Event handling
